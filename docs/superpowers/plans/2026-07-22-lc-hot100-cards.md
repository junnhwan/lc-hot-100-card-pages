# LC Hot 100 速记卡 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a polished static GitHub Pages site that turns `LC Hot 100.md` into browsable flashcards with hints, collapsible Go code, card/quiz modes, and localStorage mastery.

**Architecture:** Node build script parses Markdown + merges `notes-override.json` into `public/problems.json`. Vanilla HTML/CSS/JS SPA-less UI (Ink Studio) loads the JSON, renders sidebar + list, overlays for card/quiz. No backend; mastery only in `localStorage`.

**Tech Stack:** Node 22, native ES modules for build, vanilla JS frontend, highlight.js (CDN or vendored) for Go highlighting, GitHub Actions → Pages.

**Spec:** `docs/superpowers/specs/2026-07-22-lc-hot100-cards-design.md`

Implementation notes:

- Split or scan line-by-line / regex for `^###\s+(.+)$`

- [ ] **Step 1: Write merge tests**

Priority: MD block > override > `//` comment heuristic.

```js
import { mergeProblemNotes } from '../scripts/lib/merge-notes.mjs';

it('prefers MD hints over override', () => {
  const p = mergeProblemNotes(
    { code: '// x\nfunc f(){}', hintsFromMd: ['from md'], notesFromMd: [] },
    { hints: ['from override'], notes: ['n'] }
  );
  assert.deepEqual(p.hints, ['from md']);
  assert.deepEqual(p.notes, ['n']); // notes still from override if MD empty
});

it('extracts // comments when nothing else', () => {
  const p = mergeProblemNotes(
    { code: 'func f(){\n  // 先查后写\n  return nil\n}', hintsFromMd: [], notesFromMd: [] },
    undefined
  );
  assert.ok(p.notes.some(n => n.includes('先查后写')) || p.hints.some(h => h.includes('先查后写')));
});
```

Heuristic (spec floor): non-empty `//` lines → put into `notes` (strip `//`); if none, both arrays empty.

- [ ] **Step 2: Implement merge + build**

`merge-notes.mjs`: export `mergeProblemNotes(problemPartial, overrideEntry?)` → `{ hints, notes }`.

`build-data.mjs`:

```js
import fs from 'node:fs';
import path from 'node:path';
import { fileURLToPath } from 'node:url';
import { parseHot100Markdown } from './lib/parse-md.mjs';
import { mergeProblemNotes } from './lib/merge-notes.mjs';

const root = path.resolve(path.dirname(fileURLToPath(import.meta.url)), '..');
const md = fs.readFileSync(path.join(root, 'content/LC Hot 100.md'), 'utf8');
const override = JSON.parse(fs.readFileSync(path.join(root, 'data/notes-override.json'), 'utf8'));
const { categories, problems: raw } = parseHot100Markdown(md);
const problems = raw.map(p => {
  const { hints, notes } = mergeProblemNotes(p, override[p.id]);
  return {
    id: p.id,
    title: p.title,
    slug: p.slug,
    url: p.url,
    category: p.category,
    categoryOrder: p.categoryOrder,
    code: p.code,
    status: p.status,
    hints,
    notes,
  };
});
// recompute category counts from problems
const catalog = {
  generatedAt: new Date().toISOString(),
  categories: categories.map(c => ({
    ...c,
    count: problems.filter(p => p.category === c.name).length,
  })),
  problems,
};
const out = path.join(root, 'public/problems.json');
fs.writeFileSync(out, JSON.stringify(catalog, null, 2), 'utf8');
console.log(`Wrote ${problems.length} problems → ${out}`);
```

function loadMastery() {
  try {
    return JSON.parse(localStorage.getItem(MASTER_KEY) || '{}');
  } catch {
    return {};
  }
}
function getMastery(map, id) {
  return map[id] || 'unknown';
}
function setMastery(map, id, level) {
  map[id] = level;
  try {
    localStorage.setItem(MASTER_KEY, JSON.stringify(map));
  } catch (e) {
    console.warn('localStorage unavailable', e);
  }
}
